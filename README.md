# Taxi Booking Event Platform

This project demonstrates a scalable event-driven taxi booking system inspired by Uber/Ola architecture.

## Tech Stack
- FastAPI
- Apache Kafka
- PostgreSQL
- Docker
- Kubernetes
- Google Cloud Platform

## Microservices
- User Service
- Driver Service
- Ride Service

## Features
- User registration
- Driver registration
- Ride booking
- Event streaming using Kafka
- Microservice architecture
